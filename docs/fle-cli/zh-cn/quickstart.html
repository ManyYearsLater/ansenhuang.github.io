快速开始

快速开始

安装

$ npm install fle-cli -g

或者使用yarn来安装：

$ yarn global add fle-cli

创建工程

$ fle init <project>

本地开发

$ fle dev

线上发布

$ fle build

查看帮助

$ fle -h
