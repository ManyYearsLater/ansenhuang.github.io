本地开发

在fle项目工程的根目录下，启动本地服务来进行调试：

$ fle dev

本地开发增加了一个页面导航（index.html），方便快速跳转到指定页面，默认会构建src下所有的页面，如果需要定向编译页面，可以指定参数：

$ fle dev --target

执行以上命令来选择需要调试的页面。

如果是在手机中调试：

$ fle dev --log

执行以上命令来开启vconsole调试模式，支持查看DOM结构、网络请求、控制台输出等信息。
