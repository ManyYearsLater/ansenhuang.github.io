模块编译

模块编译

该功能是用来编译JS库、框架等模块，支持导出ES6、commonjs等，项目结构与多页面工程基本相似，可以进行本地调试、示例代码发布。

通过以下命令来进行模块编译：

$ fle lib

默认编译入口是src/common/index.js。

启用文件监听，在文件改动后会再次编译模块：

$ fle lib --watch
